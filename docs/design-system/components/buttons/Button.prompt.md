Pill-shaped CTA button in the brand's condensed display type, used for booking/contact actions.

```jsx
<Button variant="primary" size="lg">Request a quote</Button>
<Button variant="outline" as="a" href="/contact">Contact us</Button>
```

Variants: `primary` (orange, main CTA), `secondary` (red, secondary CTA), `outline` (for dark hero backgrounds), `ghost` (for light sections, low emphasis). Sizes: `sm`, `md`, `lg`.
