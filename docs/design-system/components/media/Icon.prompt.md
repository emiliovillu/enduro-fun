Lightweight line-icon set matching the Lucide style (1.8 stroke, rounded caps) — used since the brand has no icon font of its own. Substitution flagged in readme.

```jsx
<Icon name="map-pin" size={20} color="var(--accent-secondary)" />
```

Available names: mail, map-pin, instagram, menu, x, globe, chevronDown, chevron-left, chevron-right, bike, phone.
